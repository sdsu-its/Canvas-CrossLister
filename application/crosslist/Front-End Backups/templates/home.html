Instructions

To get started, you will need your API key:

  1. Click on Account on the global navigation menu.
  2. Next, click Settings.
  3. Scroll down to the Approved Integrations section.
  4. Click on Generate New Access Token.
  5. Set the Purpose and Expiration
  6. Click on Generate Token

After generating the API key you are ready to Crosslist!
